Exercise 1.1.4.

Show that the following properties are valid in any vector space \(V\text{:}\)

  1. If \(\uu+\vv=\uu+\ww\text{,}\) then \(\vv=\ww\text{.}\)

  2. For any scalar \(c\text{,}\) \(c\zer=\zer\text{.}\)

  3. If \(c\vv=\zer\text{,}\) then either \(c=0\) or \(\vv=\zer\text{.}\)

  4. For any \(\vv\in V\text{,}\) \((-1)\vv=-\vv\text{.}\)

  5. The zero vector is the unique vector such that \(\vv+\zer=\vv\) for all \(\vv\in V\text{.}\)

  6. The negative \(-\vv\) of any vector \(\vv\) is unique.
