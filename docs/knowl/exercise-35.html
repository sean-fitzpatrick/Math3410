    Suppose \(U\) and \(W\) are 4-dimensional subspaces of \(\R^6\text{.}\) What is the minimum dimension of \(U\cap W\text{?}\)

  • \(2\)

  • Good job! If \(\dim U+W = 6\) (the largest it possibly can), then \(\dim U\cap W = \dim U+\dim W-\dim (U+W) = 4+4-6=2\text{.}\) If \(U+W\) is not all of \(\R^6\text{,}\) then \(\dim (U\cap W)\) could be 3 or 4.

  • \(3\)

  • While this is a possible dimension for \(U\cap W\text{,}\) it is not the smallest possible dimension.
