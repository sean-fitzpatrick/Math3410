Moreover, TheoremÂ 2.3.2 tells us why we can only consider invertibility for square matrices: we know that invertible linear maps are only defined between spaces of equal dimension. In analogy with matrices, some texts will define a linear map \(T:V\to W\) to be invertible if there exists a linear map \(S:W\to V\) such that

\begin{equation*} ST = 1_V \quad \text{ and } \quad TS = 1_W\text{.} \end{equation*}

From this definition, one can show that \(S\) and \(T\) must be bijections, and of course, if \(T\) is a bijection (as in our definition) then we know it has an inverse. What remains to be seen is that this inverse is also a linear map.
