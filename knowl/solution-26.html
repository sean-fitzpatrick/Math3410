Solution 1.6.3.1.

\(\lbrace{{\bf u}_1, {\bf u}_2, {\bf u}_3}\rbrace\) is a linearly dependent set, then adding in \({\bf u}_4\) keeps it dependent. If \(\lbrace{{\bf u}_1, {\bf u}_2, {\bf u}_3}\rbrace\) is a linearly independent set, then consider any nontrivial solution to
\begin{equation*} a_1 {\bf u}_1+ a_2 {\bf u}_2 + a_3{\bf u}_3 + a_4 {\bf u}_4 = {\bf 0} \end{equation*}
If \(a_4=0\text{,}\) this gives us a nontrivial dependence relation for the independent set. If \(a_4\ne 0\text{,}\) the equation can be solved to show \({\bf u}_4\) is a linear combination of \(\lbrace{{\bf u}_1, {\bf u}_2, {\bf u}_3 }\rbrace\text{.}\) \(\lbrace{{\bf u}_1, {\bf u}_2, {\bf u}_3, {\bf u}_4}\rbrace\) is a linearly dependent set precisely when \(\lbrace{{\bf u}_1, {\bf u}_2, {\bf u}_3}\rbrace\) is a linearly dependent set.
