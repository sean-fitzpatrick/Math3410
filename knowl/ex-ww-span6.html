Let \(x, y, z\) be (non-zero) vectors and suppose \(w = -20x-10y+4z\text{.}\)
If \(z= 4x+2y\text{,}\) then \(w =\) \(x+\) \(y\text{.}\)
Using the calculation above, mark the statements below that must be true.
  • Span(x, y, z) = Span(w, x, y)
  • Span(w, z) = Span(y, z)
  • Span(x, z) = Span(y, z)
  • Span(w, y, z) = Span(w, z)
  • Span(w, x, z) = Span(x, y)
