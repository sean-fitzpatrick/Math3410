Let \(x, y, z\) be (non-zero) vectors and suppose \(w = -8x-4y-3z\text{.}\)
If \(z= -2x-y\text{,}\) then \(w =\) \(x+\) \(y\text{.}\)
Using the calculation above, mark the statements below that must be true.
  • Span(x, z) = Span(w, z)
  • Span(w, z) = Span(x, y)
  • Span(w, x, y) = Span(y, z)
  • Span(x, y) = Span(x, y, z)
  • Span(w, y) = Span(x, z)
