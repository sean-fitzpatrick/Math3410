Exercise 9.

If \(T: {\mathbb R}^3 \rightarrow {\mathbb R}^3\) is a linear transformation such that
\begin{equation*} T\left( {\left[\begin{array}{c} 1\cr 0\cr 0 \end{array}\right]} \right) = {\left[\begin{array}{c} -4\cr 0\cr -1 \end{array}\right]}, \quad T\left( {\left[\begin{array}{c} 0\cr 1\cr 0 \end{array}\right]} \right) = {\left[\begin{array}{c} -1\cr 3\cr 3 \end{array}\right]}, \quad T\left( {\left[\begin{array}{c} 0\cr 0\cr 1 \end{array}\right]} \right) = {\left[\begin{array}{c} -1\cr 3\cr -2 \end{array}\right]}, \end{equation*}
find the value of \(\ T \left({\left[\begin{array}{c} -1\cr 4\cr 2 \end{array}\right]}\right)\text{.}\)
