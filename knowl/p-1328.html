Using properties of the norm, we can show that this distance function meets the criteria of whatâ€™s called a metric. A metric is any function that takes a pair of vectors (or points) as input, and returns a number as output, with the following properties:
  1. \(d(\xx,\yy)=d(\yy,\xx)\) for any \(\xx,\yy\)
  2. \(d(\xx,\yy)\geq 0\text{,}\) and \(d(\xx,\yy)=0\) if and only if \(\xx=\yy\)
  3. \(d(\xx,\yy)\leq d(\xx,\zz)+d(\zz,\yy)\) for any \(\xx,\yy,\zz\)
