Exercise 5.

Suppose \(f :\mathbb{R}^2 \to \mathbb{R}^3\) is the function defined by \(\displaystyle{ f(x,y) = {\left[\begin{array}{c} 5x+3y\cr -4y+3x\cr -3x\cr \end{array}\right]}. }\)
  1. What is \(f( 2, -5 )\text{?}\) Enter your answer as a coordinate vector of the form \(\langle 1,2,3\rangle\text{.}\)
  2. If \(f\) is a linear transformation, find the matrix \(A\) such that \(f(\mathbf{x}) = A \mathbf{x}\text{.}\)
  3. Find bases (i.e., minimal spanning sets) for the kernel and image of \(f\text{.}\)
  4. The linear transformation \(f\) is:
    • injective
    • surjective
    • bijective
    • none of these
