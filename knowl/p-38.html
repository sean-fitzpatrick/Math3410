A real vector space (or vector space over \(\R\)) is a nonempty set \(V\text{,}\) whose objects are called vectors, equipped with two operations:
  1. Addition, which is a map from \(V\times V\) to \(V\) that associates each ordered pair of vectors \((\vv,\ww)\) to a vector \(\vv+\ww\text{,}\) called the sum of \(\vv\) and \(\ww\text{.}\)
  2. Scalar multiplication, which is a map from \(\R\times V\) to \(V\) that associates each real number \(c\) and vector \(\vv\) to a vector \(c\vv\text{.}\)
