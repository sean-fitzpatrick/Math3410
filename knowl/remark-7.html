Remark 2.2.2.

Caution: the kernel is the set of vectors that \(T\) sends to zero. Saying that \(\vv\in \ker T\) does not mean that \(\vv=\zer\text{;}\) it means that \(T(\vv)=\zer\text{.}\) Although it’s true that \(T(\zer)=\zer\) for any linear transformation, the kernel can contain vectors other than the zero vector.
In fact, as we’ll see in Theorem 2.2.11 below, the case where the kernel contains only the zero vector is an important special case.
