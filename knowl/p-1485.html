Let \(T:V\to V\) be a linear operator. Then:

  1. \(\displaystyle \{\zer\}\subseteq \ker T \subseteq \ker T^2 \subseteq \cdots \subseteq \ker T^k\subseteq \cdots\)

  2. If \(\ker T^{k+1}=\ker T^k\) for some \(k\text{,}\) then \(\ker T^{k+m}=\ker T^k\) for all \(m\geq 0\text{.}\)

  3. If \(n=\dim V\text{,}\) then \(\ker T^{n+1} = \ker T^n\text{.}\)
