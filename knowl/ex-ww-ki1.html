(b) Part 2: Show you know definitions.
Suppose \(A \in M_{n,k}(\mathbb{R})\text{.}\) Write \(A = (\mathbf{a_1} \mid \cdots \mid \mathbf{a_k})\text{.}\)
  1. A vector \(\mathbf{x} \in \mathbb{R}^k\) is in the null space of \(A\) if (select all that apply):
    • \(\mathbf{y} = A \mathbf{x}\) for some \(\mathbf{x} \in \mathbb{R}^k\text{.}\)
    • The dot product of \(\mathbf{x}\) with every row vector from \(A\) is \(0\text{.}\)
    • \(\mathrm{rref}(A \mid \mathbf{y})\) has no rows of the form \((0 \ldots 0 \mid 1)\text{.}\)
    • \(A \mathbf{x} = \mathbf{0}_n\text{.}\)
    • \(\mathbf{y} = x_1 \mathbf{a_1} + \cdots + x_k \mathbf{a_k}\) for some \(\langle x_1, \ldots, x_k \rangle \in \mathbb{R}^k\text{.}\)
    • \(\mathbf{y} \in \mathrm{span} \lbrace \mathbf{a_1}, \ldots, \mathbf{a_k} \rbrace\text{.}\)
    • \(\displaystyle x_1 \mathbf{a_1} + \cdots + x_k \mathbf{a_k} = \mathbf{0}_n\)
  2. A vector \(\mathbf{y} \in \mathbb{R}^n\) is in the column space of \(A\) if (select all that apply):
    • \(A \mathbf{x} = \mathbf{0}_n\text{.}\)
    • The dot product of \(\mathbf{x}\) with every row vector from \(A\) is \(0\text{.}\)
    • \(\mathbf{y} = A \mathbf{x}\) for some \(\mathbf{x} \in \mathbb{R}^k\text{.}\)
    • \(\mathrm{rref}(A \mid \mathbf{y})\) has no rows of the form \((0 \ldots 0 \mid 1)\text{.}\)
    • \(\mathbf{y} = x_1 \mathbf{a_1} + \cdots + x_k \mathbf{a_k}\) for some \(\langle x_1, \ldots, x_k \rangle \in \mathbb{R}^k\text{.}\)
    • \(\mathbf{y} \in \mathrm{span} \lbrace \mathbf{a_1}, \ldots, \mathbf{a_k} \rbrace\text{.}\)
    • \(\displaystyle x_1 \mathbf{a_1} + \cdots + x_k \mathbf{a_k} = \mathbf{0}_n\)
