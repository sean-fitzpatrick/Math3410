In Exercise 3.1.13 we saw that the set
\begin{equation*} \{(1,0,1,0), (-1,0,1,1), (1,1,-1,2),(1,-6,-1,2)\} \end{equation*}
is orthogonal. Since it’s orthogonal, it must be independent, and since it’s a set of four independent vectors in \(\R^4\text{,}\) it must be a basis. To get an orthonormal basis, we normalize each vector:
\begin{align*} \hat{\uu}_1 \amp = \frac{1}{\sqrt{1^2+0^2+1^2+0^2}}(1,0,1,0) = \frac{1}{\sqrt{2}}(1,0,1,0)\\ \hat{\uu}_2 \amp = \frac{1}{\sqrt{(-1)^2+0^2+1^2+1^2}}(-1,0,1,1,) = \frac{1}{\sqrt{3}}(-1,0,1,1)\\ \hat{\uu}_3 \amp = \frac{1}{\sqrt{1^2+1^2+(-1)^2+2^2}}(1,1,-1,2) = \frac{1}{\sqrt{7}}(1,1,-1,2)\\ \hat{\uu}_4 \amp = \frac{1}{\sqrt{1^2+(-6)^2+(-1)^2+2^2}}(1,-6,-1,2) = \frac{1}{\sqrt{42}}(1,-6,-1,2)\text{.} \end{align*}
The set \(\{\hat{\uu}_1,\hat{\uu}_2,\hat{\uu}_3,\hat{\uu}_4\}\) is then an orthonormal basis of \(\R^4\text{.}\)
