Exercise 4.

Suppose \(A \in M_{2,3}(\mathbb{R})\) is a matrix and
\begin{equation*} A \mathbf{e_1} = {\left[\begin{array}{c} -3\cr -2 \end{array}\right]}, A \mathbf{e_2} = {\left[\begin{array}{c} -3\cr -1 \end{array}\right]}, \text{ and } A \mathbf{e_3} = {\left[\begin{array}{c} -3\cr 0 \end{array}\right]}. \end{equation*}
  1. What is \(A( -5 \mathbf{e_1} - 3 \mathbf{e_2} - 4 \mathbf{e_3})\text{?}\)
  2. Find the matrix for the linear transformation \(f\) (relative to the standard basis in the domain and codomain). That is, find the matrix \(A\) such that \(f(\mathbf{x}) = A \mathbf{x}\text{.}\)
  3. Find a formula for the linear transformation \(f\text{.}\)
  4. Find bases (i.e., maximal independent sets) for the kernel and image of \(f\text{.}\)
  5. The linear transformation \(f\) is:
    • injective
    • surjective
    • bijective
    • none of these
